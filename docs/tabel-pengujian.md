### Threshold (Batas Keberhasilan)

| Metrik                       | Smoke      | Load       | Stress     | Soak       |
| ---------------------------- | ---------- | ---------- | ---------- | ---------- |
| Error Rate (http_req_failed) | < 1%       | < 5%       | < 10%      | < 5%       |
| Response Time p(95)          | < 3.000 ms | < 5.000 ms | < 8.000 ms | < 5.000 ms |

### Kesimpulan

Sistem berhasil memenuhi semua threshold pada ketiga skenario yang telah dijalankan:

-   **Smoke Test**: Sistem berjalan normal; seluruh 13 halaman merespons HTTP 200, tanpa error.
-   **Load Test**: Pada 10 pengguna konkuren, sistem sangat responsif dengan rata-rata 59 ms dan p(95) 114 ms — jauh di bawah batas 5.000 ms.
-   **Stress Test**: Pada beban puncak 50 pengguna konkuren, sistem tetap stabil dengan 0% error. Waktu respons p(95) mencapai 4,94 detik yang masih di bawah threshold 8.000 ms.
